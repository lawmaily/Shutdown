# Shutdown Timer Application

## Overview

The Shutdown Timer application is a simple and user-friendly program that allows users to schedule the shutdown of their system after a specified delay. This utility is ideal for scenarios where users need to automate the shutdown process, such as during file downloads or system updates.

---

## Features

- **Set Shutdown Timer:** Specify the delay before the system shuts down.
- **Pause and Resume:** Cancel the countdown and resume it whenever needed.
- **Dynamic Timer Control:** Restart or modify the timer without restarting the application.

---

## How It Works

1. **Launch the Application:** Start the program to access the main interface.
2. **Set Timer:** Input the desired delay and press the Start button.
3. **Pause/Resume:** Use the Cancel button to temporarily stop the timer and the Start button to refresh it.
4. **Track Timer Progress:** Monitor the timer status using the "Timer" checkbox on the interface.
5. **Automatic Shutdown:** When the timer reaches zero, the application will initiate a system shutdown.

---

## Requirements

- **Java Version:** Ensure Java 8 or higher is installed on your system.
- **Operating System:** Compatible with Windows.
